# SparkFrame AI Creator Summit

A responsive one-page React + Vite site for an AI creator summit landing page.

## Features

- Fixed top navigation
- One-click light and dark mode
- Multiple interactions: hover feedback, click toast, schedule switching, collapsible resources, FAQ accordion, clickable speaker cards
- Mobile and desktop responsive layout
- AI-generated hero artwork integrated into the page

## Local Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deployment

This repository includes a GitHub Pages workflow that builds the site and deploys the `dist` output automatically on pushes to `main`.
